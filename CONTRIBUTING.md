# Contributing to llingr-pgx

We welcome issue reports, bug fixes and pull requests.

Before proposing a large change, please open an issue to discuss it first.
The library is deliberately small: connections, migrations, role templating,
SQL fragments; no ORM. Changes outside that scope are unlikely to be accepted.

Before submitting a pull request include tests for your contribution and
run `make all`.

For security vulnerabilities: see [SECURITY.md](SECURITY.md).
