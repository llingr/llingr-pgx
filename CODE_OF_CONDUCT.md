# Code of Conduct

Everyone participating in this project - contributors,
maintainers, and users - is expected to:

- Be respectful and constructive.
- Assume and engage in good faith.
- Focus criticism on ideas and code, not people.

The following are not tolerated:

- Harassment, discrimination, or personal attacks.
- Inappropriate language or imagery in project spaces.
- Sustained disruption of discussions or contributions.
- Publishing private information without consent.

To report a concern, email conduct@llingr.dev
